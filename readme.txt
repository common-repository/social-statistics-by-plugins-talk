Find our your most popular posts by getting the social statistics for each and every post.

== Description ==

Social Statistics is a free plugin which display the social shares for each of your posts. Social Statistics is used in the admin section of your WordPress blog. For each post the plugins shows (which you can sort in ASC/DESC order) :

*   Post’s Id
*   Post’s Title (with its hyperlink)
*   Facebook Likes (count)
*   Facebook Shares (count)
*   Facebook Comments (count)
*   Twitter Tweets (count)
*   Pinterest pins (count)
*   LinkedIn Shares (count)

Based on these social statistics you can find out the most popular post among your users, which post is most shared by your facebook users or twitter users and can also find the least popular post. So you can now actually fill up your most popular articles section based on your analysis.

For more information visit: PluginsTalk.com or dev.PluginsTalk.com

== Installation ==

This section describes how to install the plugin and get it working.

Follow these steps for manual installation:

1. Upload `social-statistics-by-plugins-talk.zip` to the `/wp-content/plugins/` directory
2. Extract the .zip file so that it creates `social-statistics-by-plugins-talk` folder in `/wp-content/plugins/` directory
3. Activate the plugin through the `Plugins` menu in WordPress

Follow these steps for installation within WordPress:

1. Goto `add new` section in `Plugins` menu inside WordPress
2. Search `PluginsTalk` in the search box
3. After finding `Social Statistics`, click `install now` from the plugin
